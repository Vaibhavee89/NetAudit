Quick Start Guide
Getting Started
  1. Start with a Network Scan to discover live hosts
  2. Perform Port Scans on discovered hosts
  3. Use OS Detection to identify system types
  4. Run Vulnerability Scans for security assessment
  5. Download detailed reports for documentation
Best Practices
  • Always obtain proper authorization before scanning
  • Start with less intrusive scans (ping sweeps)
  • Use appropriate scan timing to avoid detection
  • Document all findings in generated reports
  • Follow responsible disclosure for vulnerabilities
